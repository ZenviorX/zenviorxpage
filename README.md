# ZenviorX.github.io

我的个人主页。穿梭于代码与网络安全之间，也喜欢用镜头记录生活里的光影和风景。

## 技术栈

- **框架：** Next.js 16 + React 19
- **语言：** TypeScript 5
- **样式：** Tailwind CSS 4
- **动画：** Framer Motion
- **部署：** GitHub Pages（静态导出）

## 启动

```bash
pnpm install
pnpm dev      # 开发模式
pnpm build    # 构建静态导出（out/）
pnpm start    # 预览构建结果
```

## 页面

- `/` — 首页，个人介绍 + 特色卡片 + 站点状态
- `/walkway` — 光影栈道，摄影作品展示 + 时间线

## 部署

基于 GitHub Actions 自动构建并部署到 GitHub Pages。
